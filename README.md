# ¿En qué consiste el proyecto?
Nuestra aplicación Aquanetix tiene como objetivo principal recolectar el agua en estado deficiente y estudiarla para poderse reutilizar y así ayudar a los sectores que más la necesiten. Esta propuesta irá dirigida principalmente a empresas prestadoras de servicios de agua y alcantarillados, como SEDAPAL; y a empresas operadoras de residuos de sólidos, como RESITER; ya que ambas poseen la infraestructura logística y los canales de distribución necesarios para escalar sistemas de tratamiento a gran escala.

# Características principales:
Entre los principales features, se incorporarán tecnologías IoT, tales como:

> * Sensores que midan el PH del agua y midan si puede ser consumible o no.
> * Detectores de gases tóxicos.
> * Detección de obstrucciones en base a monitoreo de flujo.

Asimismo, para generar ingresos monetarios con la aplicación, incluiremos tres planes de suscripción para nuestros usuarios:

> * **El Plan Monitoreo Base** funciona como una puerta de entrada para proyectos piloto o pequeñas plantas que requieren supervisión esencial sin complicaciones. Se centra en la conectividad de hasta 5 sensores y la emisión de alertas críticas inmediatas ante niveles anómalos de PH o presencia de gases, permitiendo que el operador actúe de forma reactiva pero rápida para evitar sanciones ambientales o riesgos de salud básicos.
> * **El Plan Gestión Operativa** escala la solución hacia un nivel profesional, ideal para empresas de servicios públicos que manejan múltiples puntos de recolección. Este nivel no solo aumenta la capacidad de sensores, sino que introduce inteligencia de datos con análisis predictivos para anticipar picos de contaminación y un módulo de gestión de cuadrillas para remover desechos sólidos u obstrucciones de forma eficiente, optimizando así los costos de mantenimiento preventivo.
> * **El Plan Smart City / Industrial** representa la visión completa de economía circular, diseñado para grandes organizaciones o gobiernos que buscan un impacto social y ambiental medible. Su principal diferenciador es el módulo de logística para la redistribución del agua recuperada hacia sectores necesitados, integrando los datos mediante APIs a sistemas externos y ofreciendo soporte de alta prioridad para garantizar que el ciclo de reutilización nunca se detenga.
